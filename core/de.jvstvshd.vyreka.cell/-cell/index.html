canBeAccessedFrom

abstract fun canBeAccessedFrom(other: Cell): Boolean

Whether this cell can be accessed from the given location. This is used to determine whether a path can be constructed from the given location to this cell. This implies that if the other cell is not accessible at all (e.g. cell.CompetitionCell.WallCell), this method may return true as this cell is not fully seperated by walls or is an entire wall itself. This behavior should not occur as a full wall cell cannot be the start/end of a path so other may never be a wall cell.

abstract fun getNeighbors(mode: CellAccessMode = CellAccessMode.ACCESSIBLE, includeDiagonals: Boolean = false): List<Cell>

Computes all neighboring cells. That are cells for which only one dimension differs by exactly 1. Excluded are diagonal cells that will be included in the returned list if includeDiagonals is true. If this cell is located on the edge of the map, the returned list will not contain cells that are outside of the map (for example if this cell has the location 0, 0 (y coordinate is omitted), the returned list will not contain the cell at 0, -1).
